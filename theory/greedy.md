# Greedy Algorithm (탐욕 알고리즘)
- greedy는 '탐욕스러운, 욕심많은'이란 뜻
- 선택의 순간마다 당장 눈 앞에 보이는 최적의 상황만을 쫓아 최종적인 해답에 도달하는 방법
- 최적해를 구하는 데에 사용되는 근사적인 방법
 

- greedy 문제를 풀 때 타임라인에 대한 문제가 나오면 
  - 시간을 선으로 그려서 풀면 이해하기 쉽다
