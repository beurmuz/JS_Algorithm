# 아스키 코드 (ASCII)

- python에서 사용 가능한 모든 문자들은 전부 하나의 숫자와 대응된다.

## 1. `ord()` - 문자를 통해 아스키 코드 값 알아내기

- 'A'의 아스키 코드 값은 65이다.

```py
>> print(ord('A'))

65
```

## 2. `chr()` - 아스키 코드 값을 통해 문자 알아내기

```py
>> print(chr(65))

'A'
```

- `ord()`와 `chr()` 응용하기

```py
x = 'A'
ord(x)          -> 65
ord(x) + 1      -> 66
chr(ord(x) + 1) -> 'B'
```
